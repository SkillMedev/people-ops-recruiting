# People Ops & Recruiting Toolkit

**Hire, onboard, and run a team fairly — without a full HR department.** — built in-house by [Skill&nbsp;Me](https://skillme.dev).

The employer side of hiring, built fair by default. Write inclusive job descriptions and scrub them for bias, turn JDs into weighted scorecards and structured STAR question kits, personalize candidate outreach, draft careful offers and rejections, build 30/60/90 onboarding plans, and synthesize interview debriefs. Every skill keeps a human as decision-maker with explicit fairness bounds.

⭐ **If this is useful, star the repo** — it's how we gauge what to build next.

## Install

- **From the catalog:** [skillme.dev/pack/people-ops-recruiting](https://skillme.dev/pack/people-ops-recruiting) — install the whole pack into Claude in one step.
- **With the skills CLI:** `npx skills add aouellets/people-ops-recruiting`
- **Manually:** copy any `skills/<slug>/SKILL.md` into your Claude skills directory.

## Skills in this pack

- **[Job Description Writer](skills/job-description-writer/SKILL.md)** — Drafts an inclusive, role-relevant job description from a role brief and leveling.
- **[JD Bias Scrubber](skills/jd-bias-scrubber/SKILL.md)** — Flags exclusionary, gendered, age-coded, and ableist language in a job description and suggests neutral alternatives.
- **[Screening Rubric Builder](skills/screening-rubric-builder/SKILL.md)** — Turns a job description into a structured, weighted candidate scorecard that scores job-relevant skills instead of vibes.
- **[Interview Question Kit](skills/interview-question-kit/SKILL.md)** — Generates structured behavioral (STAR) and role-specific interview questions mapped to competencies, with what good and weak answers look like.
- **[Candidate Outreach Personalizer](skills/candidate-outreach-personalizer/SKILL.md)** — Drafts personalized, respectful sourcing and InMail messages from a public profile summary.
- **[Offer and Rejection Writer](skills/offer-rejection-writer/SKILL.md)** — Drafts clear offer letters and respectful, legally-careful candidate rejection notes.
- **[Onboarding Plan Builder](skills/onboarding-plan-builder/SKILL.md)** — Generates a 30/60/90-day onboarding plan for a given role with milestones, check-ins, and ramp goals.
- **[Interview Debrief Synthesizer](skills/interview-debrief-synthesizer/SKILL.md)** — Consolidates multiple interviewer scorecards into a calibrated hire/no-hire summary that surfaces disagreement and bias risks.
- **[Hiring Scorecard Builder](skills/hiring-scorecard/SKILL.md)** — Builds a structured interview scorecard tied to role-specific signals.
- **[Feedback Writer](skills/feedback-writer/SKILL.md)** — Delivers feedback using SBI (Situation-Behavior-Impact) that is specific, actionable, and kind.
